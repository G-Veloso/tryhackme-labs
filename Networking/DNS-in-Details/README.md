# DNS in Details

## What this lab is
This lab covers the fundamentals of DNS and how domain name resolution works.

## Where this fits in cybersecurity
DNS is critical for network communication and is often abused in attacks such as phishing, DNS spoofing, and tunneling.

## Where it breaks / security risks
- DNS spoofing
- Cache poisoning
- Malicious domains

## Where this appears in real jobs
- SOC analysis
- Network troubleshooting
- Incident response
